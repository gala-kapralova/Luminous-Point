    <h2>Project Description</h2>
    <p>Luminous Point is a clicker game developed using Unity, designed specifically for Windows. Step into the shoes of a lighthouse keeper, balancing reality and illusion as you manage the lighthouse light to guide ships safely through the night.&nbsp;</p>
    <h2>Publishing</h2>
    <p>https://gk-play.itch.io/luminous-point</p>
    <h2>Technical Aspects:</h2>
    <ul>
        <li><strong>Game Engine:</strong> Unity</li>
        <li><strong>Platform:</strong> Windows</li>
        <li><strong>Development Language:</strong> C#</li>
        <li><strong>Graphics:</strong> The game features a mix of 2D sprites and animations. The lighthouse keeper and ships are designed with attention to atmospheric details, creating a sense of isolation and depth.</li>
        <li><strong>Sound:</strong> The game includes atmospheric sound effects to enhance the immersive experience.</li>
        <li><strong>UI/UX:</strong> The user interface is intuitive, featuring a circular light level gauge and interactive buttons for controlling the light and sound.</li>
        <li><strong>Gameplay Mechanics:</strong>
            <ul>
                <li>Clicker mechanics for increasing and maintaining light levels.</li>
                <li>Dynamic text and narrative elements that change based on light levels.</li>
                <li>Animated ships that pass by at different light levels, with plans for further interaction in future updates.</li>
            </ul>
        </li>
        <li><strong>Optimization:</strong> The game is optimized for smooth performance on Windows, ensuring a seamless experience even on lower-end systems.</li>
        <li><strong>Future Updates:</strong> Plans include adding new scenes and interactions, especially focusing on developing the project into a visual novel.</li>
    </ul>
